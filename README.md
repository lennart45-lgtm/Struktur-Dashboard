# ⚙️ MachBau Dashboard

Dein persönliches Student-Life-Dashboard für Maschinenbau an der HAW Kiel.

## 🚀 Deployment (10 Minuten)

### Schritt 1: GitHub
1. Geh auf https://github.com → Account erstellen (falls noch nicht)
2. Klick auf "+" → "New repository"
3. Name: `machbau-dashboard` → "Create repository"
4. Lade alle Dateien aus diesem ZIP hoch ("uploading an existing file")

### Schritt 2: Vercel
1. Geh auf https://vercel.com → "Sign up with GitHub"
2. Klick "Add New Project"
3. Wähle dein `machbau-dashboard` Repository
4. Framework: **Vite** (wird automatisch erkannt)
5. Klick **Deploy** → fertig!

### Schritt 3: Handy
1. Öffne deine Vercel-URL auf dem iPhone/Android
2. Safari: Teilen → "Zum Home-Bildschirm"
3. Chrome Android: Menü → "App installieren"
4. Das Dashboard erscheint als App auf deinem Homescreen!

## 🔄 Daten zwischen Geräten synchronisieren
→ Tab "🔄 Sync" im Dashboard benutzen

## 🛠 Lokal starten (optional)
